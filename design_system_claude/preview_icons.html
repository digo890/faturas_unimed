Iconography

DSU uses Phosphor Icons. Two weights only: Regular for all default UI, Fill for active nav states and filled accents. Size via font-size; colors inherit currentColor.

Sizes

Three canonical sizes. Anything else breaks the grid.

16 · inline text
24 · default
32 · shortcuts

Weights in use

Only Regular and Fill are sanctioned. Bold / Duotone / Thin / Light require explicit design approval.

ph · RegularDefault — all UI
ph-fill · FillActive nav / accents
ph-boldUse only with approval
ph-duotoneAvoid — breaks rhythm

Uso em código

Phosphor entrega uma tag <i> transformada em SVG pelo script. Sempre defina font-size explícito; não altere a cor por fill — use color.

<script src="https://unpkg.com/@phosphor-icons/web@2.1.2"></script>
<i class="ph ph-heartbeat" style="font-size:24px"></i>
<i class="ph-fill ph-house" style="font-size:24px;color:var(--color-primary)"></i>
